# Move

A minimalist house moving planner, optimised for iPhone.

## Features

- **Dashboard** — Overview of money spent and money still due
- **Shopping List** — Essential and non-essential items, categorised by room (Living Room, Bedroom, Kitchen, General)
- **Expenses** — Track moving costs, utilities, deposit, insurance, and fees
- **Budget** — Set and monitor budgets per room or category

## Usage

Open `index.html` in any browser, or deploy via GitHub Pages. On iPhone, tap Share > Add to Home Screen for a full-screen app experience.

All data is stored locally on your device.

## Tech

Single-file React PWA. No build step required.
